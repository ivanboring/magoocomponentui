Magoo Component UI · Theme directions

One CSS-less markup, four token worlds

Every component ships zero CSS — just tokenized Tailwind utilities. These four themes redefine the same variable names (--color-primary, --radius-card, --shadow-card…) with different values. Swap a value block and the whole library re-skins. Each panel below is rendered entirely in its own tokens.

Colour, radius, shadow and spacing are exact. Display faces show as system fallbacks here — the intended face is labelled per panel and loads for real in the live preview.
